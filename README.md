## 5. 진행 방식

### 🧑‍💻 각자 사전 준비

- 해당 주차 범위에 대해 다음을 자유롭게 기록합니다:
  - 개념 요약
  - 이해 안 되는 부분
  - 추가로 학습한 내용
 
- **기록 형식 자유 / 분량 제한 없음**
- 기록할 내용이 없으면 작성 생략 가능

### 🗣️ 만났을 때

- 자유로운 흐름 속에서 이야기 나누기
  - 직접 설명하고 싶은 내용
  - 의문이 생긴 포인트
  - 토론하고 싶은 내용
